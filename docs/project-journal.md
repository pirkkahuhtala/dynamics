# Project Journal

> Chronological log of all project activity. Newest entries first.
> Every session, decision, probe, assessment, mob session, and pivot is recorded here.

---

## Entry Format

```
### YYYY-MM-DD — [Event Type]

**Cynefin context:** [Domain / Inflection point / Dynamic line]
**Participants:** [Agents/perspectives involved]

[Description of what happened, what was decided, what was learned]

**References:**
- [Link to ADR, probe, or other artifact]
```

## Event Types

- `Assessment` — Cynefin domain evaluation via `/assess`
- `Probe Designed` — New safe-to-fail experiment via `/probe`
- `Probe Completed` — Experiment results recorded
- `Validation` — Inflection point check via `/validate`
- `Decision` — ADR recorded via `/decide`
- `Mob Session` — All-perspective evaluation via `/mob`
- `Retrospective` — Project review via `/retro`
- `Pivot` — Red Line dynamic triggered via `/pivot`
- `Transition` — Area moved between Cynefin domains
- `Session` — Work session start/end
